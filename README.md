# Chat Anchors (对话导航)

一个轻量级的浏览器插件，专为 **ChatGPT** 和 **Google Gemini** 设计。它能自动生成对话目录锚点，帮助你在长对话中快速定位、跳转和预览内容。

## ✨ 主要功能

* **自动生成目录**：自动识别页面中的用户提问，生成悬浮的 3D 滚动目录。
* **双向定位**：
* 点击目录 -> 页面平滑滚动到对应对话。
* 滚动页面 -> 目录自动高亮当前视野内的对话。


* **智能预览**：鼠标悬停在锚点上时，会自动弹出气泡预览完整的提问内容（智能边缘检测，不遮挡视线）。
* **沉浸式体验**：
* 支持窗口拖拽移动、双向拉伸调整大小。
* **管理功能**：支持“隐藏”不需要的锚点，被隐藏的锚点会收纳在底部折叠栏中，随时可撤销。

## 🚀 安装方法

### 方式一：作为油猴脚本 (Tampermonkey) 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 创建一个新脚本。
3. 将 `index.js` (或完整代码) 复制粘贴到编辑器中。
4. 确保 `@match` 包含以下规则：
```javascript
// @match https://chatgpt.com/*
// @match https://chat.openai.com/*
// @match https://gemini.google.com/*

```


5. 保存并启用。

### 方式二：作为 Chrome 扩展程序加载

1. Clone 本仓库。
2. 在 Chrome 地址栏输入 `chrome://extensions/`。
3. 开启右上角的 **"开发者模式"**。
4. 点击 **"加载已解压的扩展程序"**，选择该文件夹即可。

## 📖 使用说明

1. **打开面板**：进入 ChatGPT 或 Gemini 对话页面，面板会自动出现在右侧。
2. **滚动浏览**：在面板上滚动鼠标滚轮，查看长对话的历史节点。
3. **跳转**：点击任意锚点，页面即刻跳转。
4. **隐藏/恢复**：点击锚点右侧的 `×` 可将其隐藏；点击底部“已隐藏”栏展开后，点击 `↩` 箭头即可恢复。
5. **调整布局**：拖拽面板标题栏可移动位置；拖拽面板边缘可调整宽高。
